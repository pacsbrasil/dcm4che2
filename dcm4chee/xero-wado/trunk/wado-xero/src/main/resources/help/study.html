Study XML Service

Overview

C-FIND metadata service that will return a patient root hierarchy of results in a specified text format. Constraints to the C-FIND are applied by adding parameters that correspond to the DICOM tag names. As with any C-FIND query, constraints on the containing types can be applied as well as the requested level.

An empty result set will be returned if query constraints are not provided.

See Patient level information for included query constraints/results at the patient level.

Parameters

Parameter Description Notes
AccessionNumber Constrain by the accession number of the study. Multiple values separated by \ are permitted.
studyUID Constrain by the UID of the study. Multiple entries can be searched for by separating them with the backslash \ character.
ModalitiesInStudy Constrain by the modalities in the study. Multiple entries can be searched for by separating them with the backslash \ character.
StudyDescription Constrain by the study description Allows wildcards
StudyDateTime Search by the study date-time. See Common for format.
StudyID Search by the Study ID. This is another, typically internal ID for studies.
ReferringPhysicianName Search by referring physician.
StudyStatusID For some PACS systems, studies are searchable by the study status. Impax Data Center is not one of the systems searchable in this fashion.

Return Attributes

XML Attribute Description Notes
InstanceAvailability Determines how fast it is to access the given study. ONLINE means the study is easily accessed, NEARLINE means a slight delay and OFFLINE means someone needs to manually mount a tape which could take hours to days.
NumberOfStudyRelatedSeries Contains the number of sets of images (series) that this study contains. A series is typically a set of related images all taken at once such as a pass through a CT scanner.
NumberOfStudyRelatedInstances Contains the number of DICOM objects this study contains. This will typically be an indicator of study size, but for multi-frames it may not be that good an indicator.
