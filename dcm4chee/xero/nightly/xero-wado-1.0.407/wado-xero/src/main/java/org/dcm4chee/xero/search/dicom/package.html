Web Service DICOM C-Find Search

This package provides a web interface to the DICOM C-Find searches. This allows access to web clients of the search results without needing to create an AE for every web client, and without needing generic DICOM access to the server.

The general design is quite simple - the existing C-Find search mechanism for XML is used, except that instead of converting internally to an object, the result is just directly returned. This, then uses exactly the same search criteria and columns that are available to the XML web study services.

Usage

An example usage would be: 
http://HOSTNAME/wado2/cfind?studyUID=1.2.3&level=study
