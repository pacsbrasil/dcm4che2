WADO Image Retrieval Service

Overview

Standards compliant implementation of the DICOM WADO specification (see Part 18: Web Access to DICOM Persistent Objects (WADO)).

Parameters

Parameter Description Notes
requestType Defines the type of request to perform Required. Use WADO for this service
studyUID Constrain by the study UID of the requested study Required
seriesUID Constrain by the series UID of the requested series Required
objectUID Constrain by the object UID of the requested DICOM object Required
rows Scale the image upto the indicated number of rows Optional
columns Scale the image upto the indicated number of columns Optional
help Whether to show this help message. Defaults to false
