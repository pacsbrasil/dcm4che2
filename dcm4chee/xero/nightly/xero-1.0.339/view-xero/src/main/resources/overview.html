Xero View Templates

Contains the general view templates that define the visual appearance of Xero.

html

The html templates handle standard HTML macros to display things like menus, tables etc.

ie

The ie extensions of the html templates modify standard templates to handle ie specific changes.

nojs

The nojs extensions of the html templates modify the support to enable better javascript disabled versions of the html. Both of ie and nojs can be combined for ie disabled javascript browser display.

series

The series templates handle display of the series tray and selection.

Static Data Files

There are some number of static data files used to define things that are currently static, but may in the future have additional behaviour or the ability to change the data based on preferences, site setup etc.
